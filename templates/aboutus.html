About Us

Hi, I'm Tinkal Kumar and I am the sole developer of this Loan Prediction with EMI Calculator project. This project is a part of my college project and is aimed to provide users with an easy-to-use tool to calculate EMI for their loans and also get a prediction on whether their loan will be approved or rejected based on the calculated EMI.

I developed this project using Python programming language and Flask web framework. The Loan prediction project used machine learning algorithms,and EMI calculator relies on a simple set of rules to determine loan status based on the calculated EMI.

I hope you find this tool useful and if you have any suggestions or feedback, feel free to contact me at [tinkal7549@gmail.com].
